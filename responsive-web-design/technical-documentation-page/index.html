A common sight on speedrunner's livestreams is an input display. Designed to show the state of the speedrunner's gamepad, the input display is surprisingly easy to implement in many streaming setups.

D-Pad

The D pad provides digital input in 4 directions:

  • Up
  • Down
  • Left
  • Right

Most controllers allow multiple directions to be held simultanously. Some prevent opposing directions to trigger at the same time.

if d-pad-up AND d-pad-left then MOVE("Up-Left")

Analog Sticks

Two analog sticks are usually included on a gamepad. These provide the precision input required for many modern games.

if axis=="Left Stick" then MOVE
if axis=="Right Stick" then AIM

Shoulder Buttons

On the top of the controller, shoulder buttons can be found. These come in two varieties:

  • Analog "triggers"
  • Digital "bumpers"

These buttons are easier to access while using the D-Pad or Analog Sticks

if button=="Right Trigger" then SHOOT
if button=="Left Trigger" then FOCUS

Main Buttons

The main buttons are usually found on the right side of the controller.

These buttons usually directly interact with the game.

if button=="A" then YES
if button=="B" then NO

Other Buttons

Most controllers also have a few other buttons, sometimes in the center of the controller.

These buttons can perform meta functions, such as pausing the game.

if button=="start" then PAUSE
if button=="select" then OPEN_MENU
